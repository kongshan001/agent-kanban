# v0.7 迭代记录

## 本轮改进

### 1. 配置保护
- `kanban init` 不再覆盖已有 `kanban.config.yaml`
- 避免用户手动修改配置后被 init 重置

### 2. 任务重置命令
- `kanban task reset TK-001` — 重置任务到 DRAFT
- `kanban task reset TK-001 --force` — 同时清理 worktree 和分支
- 解决中断后残留 worktree 的问题

### 3. Skill 文档完善
- SKILL.md 重写，包含完整的安装方式（pip/git/Claude Code Skill）
- 命令参考表格
- 配置说明
- Agent 支持矩阵

### 4. Hermes 输出解析优化
- `_clean_hermes_output` 改为取最后一个 "## 评估报告" 之后的内容
- 过滤 prompt 模板残留和 hermes 结束标记
